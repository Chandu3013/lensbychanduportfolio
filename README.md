# Lensbychandu Portfolio 📸

A stunning Next.js and Tailwind CSS portfolio ready to showcase your photography skills and capture new clients.

## 🚀 Quick Start & Deployment

This project is perfectly set up to be deployed instantly on Vercel.

### 1. Test Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### 2. Customizing Your Content

- **Photos**: Place your high-res photos in the `public/images/` folder. Update the image paths in `components/Hero.js`, `components/GalleryGrid.js`, and `pages/gallery.js` to point to your new files (e.g., `"/images/my-photo.jpg"`).
- **Styling**: Tweak `styles/globals.css` or the Tailwind classes in your components if you want to alter the color palette or fonts. The portfolio defaults to a premium cinematic dark mode.

### 3. Deploy to Vercel (Auto-Deployment)

1. Push this entire folder/repository to your own GitHub account.
2. Sign up or log into [Vercel](https://vercel.com).
3. Click "Add New Project" and import the repository you just pushed.
4. Allow Vercel to auto-detect the Next.js framework.
5. Click **"Deploy"**!

Within minutes, you'll have a live, blazing-fast portfolio sharing your creative vision with the world.
